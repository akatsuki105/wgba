# nextjs-starter

## features

- webpack5
- tailwind.css + styled-components (+ twin.macro)
- useful atoms components

## start

```sh
yarn install
yarn dev
```

## update dependencies

```sh
python3 ./scripts/yarnd.py
```

## hygen

**Create pages**

```sh
hygen pages new
```

**Create container component**

```sh
hygen container new
```

**Create presentational component**

```sh
hygen present new
```
